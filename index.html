经过IE6&IE6+，chrome，firefox测试兼容使用方法： {yourId}替换成你的id

HTML

  1. <div class="{yourId}">
  2.     <div class="{yourId}_menu">
  3.         <a href="#">导航一</a>
  4.         <a href="#">导航二</a>
  5.         <a href="#">导航三</a>
  6.         <a href="#">...</a>
  7.     </div>
  8.     <div id="{yourId}_animate_block"></div>
  9. </div>

CSS以下这些参数可以根据需要自行调整

  1. #nav{ position:relative; width:px; height:70px; padding-bottom:10px; background:#fff}
  2. #nav #nav_menu{ height:50px;}
  3. #nav #nav_menu a{ display:block; float:left; height:50px; padding:0 33px; line-height:50px; color:#666; font-size:18px; text-decoration:none;}
  4. #nav #nav_animate_block{ position:absolute; top:60px; height:2px; overflow:hidden; background:#80b600;}

JS

  1. <script type="text/javascript" src="lib/jquery-1.9.1.js"></script>
  2. <script type="text/javascript" src="jquery.bouncemenu.js"></script>

调用

  1. $("#{yourId}").bouncemenu({
  2.     'nav_current' : {nav索引}, //设置当前nav，{nav索引}替换成你想要的第N个nav的索引(n-1)，默认值是0,即第一个
  3. });
